# SHT40I_AD1F を使った温湿度表示器 v1.0

秋月で DIP 化したものが売っている。

https://akizukidenshi.com/catalog/g/g130207/

MCU は RL78/G24 R7F101G6E2DSP ボード v1 としてみる。

[回路図](./v1.0/sht4xi_v1.0/sht4xi_v1.0.pdf)

I2C 親機受信をしたことが無いので、上手くプログラムを動かせられなかったら ATMega328P
(Arduino) に戻ろう。

[設計図](./v1.0/SHT4XI1.0.pdf)

DIP と SIP の違いが分かってないまま制作したので、間違ってるけど放置。
