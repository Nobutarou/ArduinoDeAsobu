# ピンの研究

データシートと参考回路を見ながら

| ピン番号 | 記号 | コメント |
| --- | --- | --- |
| 1 | VDD25OUT  | 内蔵 2.5V LDO の出力ピンで、参考回路には 0.1uF と 4.7uF。データシートには 22uF 付けろとあるが USB にそんなもん付けるべきかは疑問。まあ手持ちのセラコンとケミコンを適当に付けておくことにする |
| 2 | VSSREG    | 内蔵 2.5V LDO の VSS で参考回路は、そのまま GND |
| 3 | LED4 | 取り外し可否の入力にも使えるようだが、データシートに詳細の記載がない。参考回路は LED 点灯にしか利用していない。使わないつもり |
| 4 | LED3 | 同上 |
| 5 | LED2 | 同上 |
| 6 | LED1 | 同上 |
| 7 | GREEN | LED を光らせるための 3.3V 出力。使わないつもり |
| 8 | AMBER | LED を光らせるための 3.3V 出力かつ、光らすポートを指示する入力でもあるようだが、参考回路は 3.3V 出力のみ。なので使わないつもり |
| 9 | VDD33 | 3.3V 入力で参考回路は 3.3V レギュレータ出力を入れている。良く見たら内蔵 3.3V LDO が付いていない。まじか |
| 10 | X1 | 水晶30MHz。リード部品が見つからないがトヨコムの FA-238 30MHz が RS で売っている |
| 11 | X2 | 同上 |
| 12 | VDD25 | 参考回路は #1 VDD25OUT と繋いでいないがデータシートは付けろと書いてある。内部で繋がっているということだろうが、普通はそんなこと知らないはずだから、繋いでおくほうが自然な気がする。コンデンサは 0.1uF とあるので適当なセラコンを |
| 13 | BUS_B | データシートは Powermode Select とあるだけで、何も説明がない。参考回路は 3.3V を入力。情報が無いのでそうするしかない |
| 14 | TEST | Test signal input としか無く不明。参考回路は GND へ。まあそうしてみるしかない |
| 15 | RREF | 2.31kΩ ±1% って少量販売している物が見つからない|

# まとめ

2.31kΩでギブアップ。情報も少ないし。
