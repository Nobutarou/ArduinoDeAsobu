注1:
[ユーザーズマニュアル](https://www.renesas.com/ja/document/mah/ra2e2-group-users-manual-hardware?r=1532431#unique_1086)
の 35.2 節では VCL に 4.7uF 付けてるけど、まあ 1.5uF でいいでしょ。他に使うわけでもないし。
同様に VCC には 0.1uF 付けてるけど、まあ 0.22uF でいいでしょ。

注2:
一応、ユーザーズマニュアルの 表36.9 で RES のリーク電流最大値は 1uA. 10kΩなら電圧変動は 0.01V。
ということで、まあ 1k - 100kΩくらいなら、何でも良いんじゃね？
MD は「その他のポート」だと思うので 1uA だから RES と同じで良いでしょう.

書き込みは

- 電源投入
- RES, MD はプルアップ
- MD をプルダウン
- RES を Low にして Hi に戻す

で書き込みモードらしい。ちょっとだるい。だるくなったので保留。
