# 動機

[a-p-prog 機](../../ArduinoPICProgrammer) の UART 接続が不安定。どうも 57600bps 接続決め打
ちのようだが、PIC16F1455 で作った UART-USB 変換器には、それを安定させる力はないようだ。

PIC16F1455 は [PS/2-USB変換器](../../PS_2-to-USB_KB/) に実際に活用してるマイコンだから、気
軽に高い確実性で書き込みできるようにしておきたい。

ということで PIC16F1455 を書き込み&電源供給とし、PIC16F1455 をターゲットとする開発ボードを
作っておくことにした。

# v1.0 (制作せず)

基本的には [pic-pic-progi v1.0](../../Pic_Pic_Prog/) から、USB-B を外して、代りに
pic-pic-prog を迎え入れるためのピンヘッダを立てておく。

[設計図](./v1.0/PIC16F1455_v1.0.pdf)
pic-pic-prog 1.0 用のアダプタも設計しておく

ここまで描いて pic-pic-prog 1.0側が、ちょっと嫌になったので保留。小くするために横幅を削っ
てるが、それのせいで USB-B 用の穴空けが難しくなってるのと、IC ソケットとピンソケットがギチ
ギチでピンソケットが開き気味になってるのが、ちょっと嫌だった。

で、これを作ってから pic-pic-prog 側を変更して、もう一回、アダプタを作り直すよりは、
ASOboard 側で CP2104 を UART にして、a-p-prog の成功確率を変えておいてから、pic-pic-prog
ごと作りなおすのが良い気がしてきた。なので ASBoard の CP2104 対応を先にやることにする。
