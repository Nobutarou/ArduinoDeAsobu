# USB Mod 3

オーストラリアのパーツショップで販売されている。オリジナル商品なのかな。

temperosystems.com.au/products/usb-mod-3/

USB type B。

普通と変わっているのは、ブレークアウト基板に近くて、USB 挿しても USB シリアルアダプタとし
て動作しない。ユーザがそこからやる必要がある。

普通の USB シリアルアダプタとして使うには、次の 3配線が必要。

- EP (#9) を RSTO (#10) 
- VIO  (#12) を +V (#13 または #4)
- PCTL (#18) を GND (#16, #1, #2, #29 のどれか)
