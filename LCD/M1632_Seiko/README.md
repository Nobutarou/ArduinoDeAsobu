# I2C アダプタ v1.0

[LM162](../LM162AT1_Sharp/) と同じアダプタを作る。Vdd と Vss が逆になっている。

[設計図](./v1.0/I2C_Adapter_1.1.pdf)

右側は詰めれば 1,2 列減らすこともできそうだが、どのみに LCD の下で隠れるし、スポンジかで
LCD の支えにすることを考えると、のり代も欲しいので、そのままにした。

C2 のコンデンサはおまけ。必要ない。消費したいだけなので、入るもので、容量が大きすぎなければ OK

普段は上向きにはソケットを付けてヘッダは避けているのだけど、LCD 単体でテストしたい場合を考
えると LCD にはソケットを付けたい。また、一回動けば、そんなに外すものでもない。というとこ
ろから、LM162 のとき同様に、ヘッダを上向きに付けることにする。
